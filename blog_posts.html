Tutorial: How To Make A Basic Website

  1. To start off your website, you first need a<html> tag. This is a basic tag that is the first line of every single html page. At the bottom of the page, close the tag by putting</html>, so the computer knows the page is done. This is also how all tags are closed in html: put a / before restating the tag name.

  2. Second, you need a <head> tag. This includes all the main elements taht go at the top of the page, such as <style> and <meta>.

  3. Third, you need a <title> tag. This actually isn't necessary, but if you want your tab text changed, this is the tag for you. It will go inside the <head> tag, as it's one of the main elements at the top of an html page.

  4. Next comes the <body> tag, the heart and soul of html. This is where all of your other tags will go, like your <h1> tag.

  5. After that, you basically have your code down. Now, you need some style. you can use style="" to reference CSS, or Cascading Style Sheets. Inside the quotes, you can put font-family, which then takes in a font name, such as "times new roman".

  6. After fonts, you need color. w3schools.com has a great color picker that you can choose background and text colors from.

  7. Now, you can add some images! This is done using the <img src=""> tag>. Inside the quotes, you put the url of the picture that you found. After the the starting tag, put the link text that you want to show on the page.

  8. You'll probably want to check your code now, just to make sure that you haven't skipped closing any tags or misspelled any text.

  9. Run the html code you've just made by right-clicking on the code, and selecting "Open in Browser".

  10. Congratulations! You've made a basic html page. But the fun doesn't end here. You can experiment with other tags in the future, but these are the basics.
