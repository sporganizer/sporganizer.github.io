How Do I Do... Everything? - 7/25/16

Today we learned HTML - Hyper Text Markup Language in web design. It was kind of overwhelming with all the new tags we were learning (like <ol>, <h1>, and <button>), but it was fun to feel like a hacker, editing all this code.

In Photoshop, we learned about all the different types of tools that you can use to edit images. We also learned about typefaces and font families, which we enjoyed. Although it was kind of difficult to understand which fonts were werif and which were sans serif, we were eventually able to grasp the concepts.

We also learned python, and by the end of class, we knew all the different data types in python.

Finally, we covered business, the core of this camp. We went over what a startup is, and how to make one. At the end, we touched on what we had to do, and I must say, it looked reallyoverwhelming.
